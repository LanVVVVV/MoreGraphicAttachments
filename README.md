# MoreGraphicAttachments

MoreGraphicAttachments is a mod for **Monster Black Market** —— adding additional Spine Attachments graphics for game character units.

### Features

- **Information Panel Optimization**
  - Change type identifiers in the information panel from letters to numbers
  - Avoid the display limit of 26 letters

- **Gallery Extension**
  - Automatically extract all vanilla attachment types
  - Allow switching between all attachment types in the gallery

- **Colored Clothes System**
  - Add a Clothes color info slot in both the information panel and the gallery
  - Allow switching Clothes colors in the information panel
  - Initialize colored Clothes when characters are generated
  - Choose mode or customize the chance of colored Clothes appearance in mod settings
  - Control whether offspring inherit Clothes color from parents

- **Info Slot Extension**
  - Add a Clothes type info slot in both the information panel and the gallery
  - Allow switching Clothes types in the information panel
  - Clothes type info slot is only displayed when the race configuration of a character includes extra Clothes types
