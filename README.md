# MoreGraphicAttachments

MoreGraphicAttachments is a mod for **Monster Black Market** —— adding additional Spine Attachments graphics for game character units. Currently, colored clothing have been implemented.

### Features

* Initialize and modify colored clothing for characters.
* Select mode or customize the chance of colored clothing appearing in the mod settings.
* Controlling whether offspring inherit clothing color.

### Requirements

| Dependency           | Version    |
| -------------------- | ---------- |
| MBM.ModLoader        | ≥ 0.6.0    |
| Monster Black Market | ≥ 2.1.2.0  |
